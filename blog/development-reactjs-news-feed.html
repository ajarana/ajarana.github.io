Building a ReactJS News Feed

This news feed was built using ReactJS and Redux. I also used plain, old CSS3 for the layout as well as icons. I didn't bother generating the header or footer with React, however, because I use that component across my whole website and it's completely static.

React

As I mentioned before, everything besides the header and footer are rendered with ReactJS. I separated the layout between presentational and container components in order to separate components that serve as sources of data versus those that are simply passed data to present.

This also served as a great introduction to ES6. My components included:

  • Filter (container component)
  • Picker (presentational component)
  • Progress Bar (presentational component)
  • Refresh (presentational component)
  • Sources (presentational component)

Redux

Redux was used for state management; it was really annoying to have to pass data through a bunch of componenets just so the last component in the chain could have access to that data. Redux stores the entire state of the application in one object so that you can access it at any one point in time.

In my project, Ajax calls are made via the Fetch API depending on the currently selected category. The returned data is stored in one object. Components like my Progress Bar can then directly access the data to give an accurate representation of how much data has been downloaded at any given point in time.

Another presentational component, the category selector, also receives data from my Filter container component in order to change its styling according to the selected category.

CSS3

The styling was very straightforward for this project. The trickiest part was making sure components were rendered with the correct CSS class.

Conclusions

I really enjoyed getting to know the basics of React and Redux here. Although I used the JSX preprocessor for React, I was also able to use vanilla JS directly alongside it and continue learning its nuances.
