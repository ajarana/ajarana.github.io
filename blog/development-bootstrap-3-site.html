Building a Bootstrap 3 Image Gallery

I designed and developed this gallery using Boostrap 3 components, LESS, HTML5, and a little bit of JavaScript. Truth be told, I did not learn as much about these technologies, excluding Bootstrap obviously, as I have from other projects where I built things completely from scratch.

Boostrap Components

Components used included navbars, containers, columns, among others. Their extensive documentation does more than enough to get you through any problems. It's a very easy framework to learn.

LESS

My LESS directory consisted of the Bootstrap LESS files along with my own custom ones. Vanilla Bootstrap isn't enough for a distinct look, though customization can alot be quite a pain since Bootstrap 3 does not use things like flexbox.

HTML5

I used the basic HTML5 template from Bootstrap's website for this part. As I stated earlier, this type of copy-paste coding isn't very conductive to learning anything, but the main goal was to learn Bootstrap. However, I did pick up some useful things from their template, such as the use of the viewport meta tag.

JS

I used a little bit of Vanilla JS to make a counter that keeps track of the date. jQuery is used for other UI effects for the header and image gallery.

Conclusions

The best learning is done building from scratch. Using only libraries and frameworks for your workflow without bothering to learn the fundamentals only does you a disservice in the long run.
